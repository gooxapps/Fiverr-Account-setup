## ✨ Features

- **🤖 AI Gig Generator**: Type a simple description of your service (e.g., "Flutter App Developer") and instantly generate all 5 Fiverr tabs perfectly optimized for SEO.
- **🔗 Competitor URL Importer**: Paste up to 3 competitor Fiverr URLs. The AI will analyze their structure, synthesize their best traits, and create the ultimate gig for you.
- **🎨 AI Thumbnail Generator**: Describe the thumbnail you want, and the AI will generate a highly professional, text-free background image perfectly sized for Fiverr (550x370).
- **🌍 1-Click Translation**: Translate your entire generated gig into Spanish, French, German, or Portuguese with one click.
- **📈 Live SEO Scoring**: Get instant feedback on your gig's SEO strength.
- **💾 Export to Markdown**: Export your generated gig directly to a `.md` file for easy copying and pasting.

## 📥 Download (Mac OS)

To install the desktop application on your Mac:
1. Go to the [Releases](../../releases) tab on this GitHub repository.
2. Download the latest `.dmg` or `.app.tar.gz` file for macOS.
3. Open the file and drag **Fiverr Gig Generator AI** to your Applications folder.

*(Note: Since this is an unsigned app, you may need to Right-Click -> Open the first time you run it).*

## 💻 Development Setup

If you want to run the project locally or contribute:

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [Rust](https://www.rust-lang.org/tools/install) (Required for Tauri Desktop builds)

### Installation
```bash
# Clone the repository
git clone https://github.com/YourUsername/fiverr-gig-generator.git
cd fiverr-gig-generator

# Install dependencies
npm install

# Run the web version in development mode
npm run dev

# Run the Desktop app in development mode
npm run tauri dev
```

### Build for Production
```bash
# Build the Mac Desktop App (.dmg / .app)
npm run tauri build
```
